# 🎲 Alias Game — Игра в слова

Простая и увлекательная командная игра, где нужно объяснять слова своей команде, не называя их напрямую.

## 🎮 Что это?

**Alias Game** — это компьютерная версия популярной настольной игры "Alias" (Элиас / Скажи иначе). 
Один игрок объясняет слова, его команда угадывает. За каждое угаданное слово команда получает очко.

## ✨ Возможности

- **2-4 команды** — играйте с друзьями
- **5 категорий слов**:
  - 🐘 Животные
  - 👨‍⚕️ Профессии
  - ⚽ Спорт
  - 🍕 Еда
  - 📚 Предметы
- **Случайная категория** — для любителей сюрпризов
- **⏱️ Таймер 60 секунд** на раунд
- **🏆 Подсчет очков** и итоговая таблица
- **💾 Сохранение настроек** игры

## 🚀 Как запустить?

### Вариант 1: Запуск через Python (для всех ОС)

**Требования:** Python 3.7 или выше

# 1. Скачайте файл alias_game.py

# 2. Запустите командой:
python alias_game.py

# Или если Python 3 установлен отдельно:
python3 alias_game.py

### Вариант 2: Готовый EXE-файл (только Windows)

**Ничего устанавливать не нужно!**

1. Скачайте файл AliasGame.exe из раздела Releases

2. Просто запустите его двойным щелчком

3. Наслаждайтесь игрой 🎉

#⚠️ Windows может попросить подтверждение запуска — нажмите "Выполнить в любом случае"
